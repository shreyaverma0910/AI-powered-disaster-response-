# AI-Powered Disaster Response System

## Project Description

This project uses Artificial Intelligence,
Machine Learning, and Computer Vision
to analyze drone footage and satellite
imagery during floods and earthquakes.

The system identifies:
- Trapped survivors
- Damaged buildings
- Flooded areas
- Safe rescue routes

and helps rescue teams respond faster
during emergencies.

---

## Features

- Disaster Detection
- Survivor Identification
- Drone Image Analysis
- Satellite Image Analysis
- Rescue Assistance
- Confidence Score Prediction
